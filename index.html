Kenapa Guru
Butuh Ini?

😩
Input Nilai Manual = Rawan Salah
Spreadsheet biasa butuh berjam-jam dan rentan salah ketik yang merusak rapor siswa.
⏰
Deadline Rapor & SKL Mepet
Cetak rapor, transkrip, SKL, dan ijazah harus selesai dalam waktu bersamaan — tanpa alat yang tepat ini sangat menyulitkan.
📡
Bergantung Internet & Server
Sekolah di daerah terpencil tidak bisa mengandalkan aplikasi berbasis cloud. Ketiga aplikasi ini berjalan penuh offline.
🔍
Kualitas Soal Tidak Terukur
Guru tidak tahu soal mana yang terlalu mudah, sulit, atau dijawab keliru oleh mayoritas siswa tanpa analisis butir soal.
😴
Presentasi Satu Arah Membosankan
Slide biasa hanya ditonton — peserta pasif dan tidak ada cara cepat untuk mengukur pemahaman kelas secara real-time.

SKL & Ijazah

Pengolah Nilai & Surat Keterangan Lulus — kelola seluruh dokumen kelulusan kelas 6 SD dari satu aplikasi: SKL, transkrip, ijazah, piagam, ranking, hingga SK Penetapan Lulus.

Dokumen yang Dihasilkan

Satu Aplikasi untuk
Semua Dokumen Kelulusan

Tidak perlu berpindah-pindah aplikasi. Semua dokumen resmi kelulusan kelas 6 dihasilkan dari satu sumber data siswa.

📋
Surat Keterangan Lulus (SKL)
Cetak SKL per siswa dengan nomor surat, tanda tangan kepsek, dan cap sekolah. Format sesuai standar dinas.
📄
Transkrip Nilai
Transkrip nilai lengkap mencakup nilai rapor, ujian sekolah, dan nilai ijazah per mata pelajaran.
🏆
Ranking Siswa
Peringkat otomatis berdasarkan jumlah nilai ijazah dari tertinggi ke terendah. Bisa dicetak sebagai lampiran.
🎖️
Piagam Penghargaan
Piagam kelulusan dengan dua pilihan desain — bawaan atau gambar custom yang bisa diunggah.
📚
Kartu Buku Siswa
Kartu buku nilai lengkap per siswa untuk arsip sekolah dan diberikan ke wali murid.
📊
Rekapitulasi Rapor
Rekap nilai seluruh siswa dalam satu tabel — mudah untuk pelaporan ke kepala sekolah atau dinas.
⚡
Laporan Kilat
Ringkasan statistik kelas: rata-rata, tertinggi, terendah, dan distribusi nilai per mata pelajaran.
✉️
Surat Undangan
Template surat undangan pengambilan ijazah/SKL dengan data siswa terisi otomatis.
📝
SK Penetapan Lulus
Surat Keputusan kepala sekolah tentang penetapan kelulusan peserta didik beserta lampiran daftar nama.
📜
Surat Kelulusan
Surat kelulusan resmi per siswa yang ditandatangani kepala sekolah, siap cetak dan diserahkan.
Alur Kerja

Dari DNS ke Ijazah
dalam 4 Langkah

1
Import DNS (Data Nominasi Siswa)
Import dari Excel Dapodik, atau input manual. Data nama, NISN, no. ijazah, dan identitas lengkap.
2
Input Nilai Rapor & Ujian
Input nilai rapor 5 semester, nilai ujian sekolah (tertulis + praktik) per mata pelajaran.
3
Hitung & Verifikasi Nilai Ijazah
Nilai ijazah dihitung otomatis dari kombinasi nilai rapor dan ujian sekolah sesuai formula yang berlaku.
4
Cetak Semua Dokumen
SKL, transkrip, piagam, ranking, SK, surat undangan — semua tercetak dari satu klik menu.

Fitur Pendukung

Lebih dari Sekadar
Cetak Dokumen

🗄️ Import Dapodik
Import Langsung dari Excel Dapodik
Upload file Excel Dapodik Kelas 6, sistem otomatis membaca data siswa dari baris header dan memfilter kelas 6 saja. Pilih data ayah atau ibu sebagai nama orang tua.
🖨️ Kop Surat Global
Atur Sekali, Berlaku Semua
Kop surat diatur sekali di menu Add-on, langsung berlaku untuk SKL, transkrip, surat undangan, SK, dan semua dokumen lainnya — tidak perlu atur ulang per dokumen.
💾 Backup & Restore
Data Aman dengan Backup JSON
Backup seluruh data ke file JSON lokal kapan saja. Restore dari backup jika terjadi masalah. File backup menyertakan timestamp otomatis.
📜
Semua Dokumen Kelulusan dari Satu Aplikasi. Tidak perlu buka Word, Excel, atau aplikasi lain. Data DNS diinput sekali, lalu SKL, transkrip, piagam, surat undangan, SK, dan semua dokumen lainnya dapat dicetak langsung — dengan format yang sudah sesuai standar dinas pendidikan.

Salindia Interaktif

Arena Presentasi Interaktif — ubah tayangan slide biasa menjadi sesi belajar dua arah dengan kuis real-time, berbagai tipe soal, dan umpan balik langsung dari seluruh peserta.

Dua Peran, Satu Sesi

Presenter & Peserta
Terhubung Real-Time

Guru masuk sebagai Presenter untuk mengelola slide dan soal. Siswa bergabung sebagai Tamu via PIN atau QR Code dari gawai masing-masing — tanpa install aplikasi.

Kendali Penuh di Tangan Guru
Kelola slide, sematkan soal per slide, mulai/akhiri sesi, pantau jawaban siswa, dan tampilkan hasil ke layar proyektor — semua dari satu dasbor.

2
Mulai Sesi — Bagikan PIN
Satu klik mulai sesi; PIN kelas dan QR Code langsung tersedia untuk peserta.
3
Buka Kuis Per Slide
Aktifkan soal kapan pun. Jawaban masuk secara real-time dan ditampilkan sebagai infografis.
4
Tampilkan Hasil ke Kelas
Grafik hasil jawaban muncul langsung di layar proyektor untuk diskusi kelas.

Electron Desktop App
Dikemas sebagai file .exe — tidak perlu instalasi Node.js atau browser terpisah. Buka dan langsung pakai.
🌐
Server Lokal Embedded
Server HTTP berjalan di dalam aplikasi — perangkat lain di LAN bisa mengakses lewat browser tanpa konfigurasi.
🔄
Sinkronisasi 5 Detik
Polling otomatis setiap 5 detik — data pendaftar, antrian, dan carousel selalu sinkron di semua perangkat.
⚛️
React + Babel In-Browser
UI dibangun dengan React (JSX dikompilasi Babel di browser) dan Tailwind CSS — tanpa build toolchain.
🔑
Aktivasi Lisensi
Sistem lisensi berbasis NPSN — satu kode per sekolah, sinkron otomatis ke semua perangkat dalam LAN.
🎠
Carousel Banner Kustom
Admin upload gambar & teks berjalan untuk halaman pendaftar — tampil di semua client secara real-time.
📋
Penerimaan Murid Baru yang Rapi, Cepat, dan Terdokumentasi. SPMB SD menggantikan antrian kertas dan formulir manual — dari panggilan nomor bersuara, formulir digital, hingga notifikasi WhatsApp ke orang tua, semua berjalan dari satu aplikasi offline yang bisa diakses seluruh panitia PPDB lewat jaringan sekolah.
